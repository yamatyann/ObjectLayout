# 接続図作成アプリ (ObjectLayout Ver2.0)

灯体（照明機材）の配置図を作成し、DMXや電源の配線計画、パッチ表などをPDF/PNGとして出力できる Windows アプリです。
照明の現場や舞台設営でのドキュメント作成を強力にサポートします。

## 🚀 What's new in Ver2.0

Ver2.0では、Python (PySide6) ベースでアプリを完全リニューアルしました。

- **⚡ 電源管理機能の追加**
  - DMX配線だけでなく、**電源ケーブルの配線**が可能になりました。
  - コンセント（回路）ごとの**使用電力（ワット数）を自動計算**し、容量オーバーを警告するレポート機能を追加。
- **↩️ Undo / Redo 対応**
  - 操作を間違えても、「元に戻す」「やり直す」が可能になり、安心して編集できます。
- **🏟️ 会場エディタ**
  - アプリ内で壁を描画したり、コンセント位置を設定したりして、オリジナルの会場データを作成・保存できます。
- **🛠️ 機材ライブラリ管理**
  - 灯体のデータをアプリ上のGUIで追加・編集できるようになりました（画像の取り込み、DMXモード設定、スナップ点設定など）。
- **📊 エクスポート機能の強化**
  - 配置図、DMX配線図、電源配線図、パッチ表、電源回路表などをまとめてPDF出力できます。
- **🔍 DMXパッチ管理**
  - アドレスの重複やユニバース超過を自動チェックする一覧ウィンドウを追加。

## 💡 どんなアプリ？

- **直感的な配置**: マウス操作で灯体を配置・移動・回転。スナップ機能できれいに並べられます。
- **配線シミュレーション**: 機材同士をクリックするだけでDMX信号線や電源線を配線できます。
- **プロパティ編集**: 選択した機材の座標、DMXアドレス、文字色などをサイドパネルで詳細に設定可能。
- **ドキュメント出力**: 現場ですぐに使える図面やリストをワンクリックで生成します。
