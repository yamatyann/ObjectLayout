# 接続図作成アプリ (ObjectLayout Ver2.00.01)

灯体（照明機材）の配置図を作成し、DMXや電源の配線計画、パッチ表などをPDF/PNGとして出力できる Windows アプリです。
照明の現場や舞台設営でのドキュメント作成を強力にサポートします。

---

## ⬇️ ダウンロード

ダウンロードは[こちら](https://github.com/yamatyann/ObjectLayout/releases/latest)

## 🚀 What's new in Ver2.00.01

Ver2.00.01では、バグの修正や軽い機能の追加しました

- **画像読み込みがうまくいかないバグ**
  - 画像読み込み時、imageフォルダ内にあるのにそれが認識されないバグを修正しました。
- **画像読み込み時のスナップ点のプレビュー**
  - 画像読み込み直後にスナップ点の修正をする際、プレビューの画像が選択前の画像になっているバグを修正しました。
- **配線時のUndo機能の追加**
  - 配線時、右クリックでUndo、Escでキャンセルに修正しました。

---

## 💡 どんなアプリ？

- **直感的な配置**: マウス操作で灯体を配置・移動・回転。スナップ機能できれいに並べられます。
- **配線シミュレーション**: 機材同士をクリックするだけでDMX信号線や電源線を配線できます。
- **プロパティ編集**: 選択した機材の座標、DMXアドレス、文字色などをサイドパネルで詳細に設定可能。
- **ドキュメント出力**: 現場ですぐに使える図面やリストをワンクリックで生成します。
